## 题目描述

>旧为了用事实说明挖掘机技术到底哪家强，PAT组织了一场挖掘机技能大赛。现请你根据比赛结果统计出技术最强的那个学校。

## 输入描述

>输入在第1行给出不超过105的正整数N，即参赛人数。随后N行，每行给出一位参赛者的信息和成绩，包括其所代表的学校的编号、及其比赛成绩（百分制），中间以空格分隔。

## 输出描述

>在一行中给出总得分最高的学校的编号、及其总分，中间以空格分隔。题目保证答案唯一，没有并列。

## 输入例子

>6<br>
3 65<br>
2 80<br>
1 100<br>
2 70<br>
3 40<br>
3 0<br>

## 输出例子

2 150
