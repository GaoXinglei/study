## 枚举
### 判断枚举是否可用
- 分析数据量 $1000ms\approx10^7$运算

    复杂度|数据量
    ---|---
    $O(n!)$|$10$
    $O(2^n)$|$20$
    $O(n^3)$|500
    $O(n^2)$|2500
    $O(nlog(n))|$10^6$
    $O(n)|10^7$
    $O(\sqrt{x})|10^14$
    $O(logn)|10^20$
    不常见等式时间复杂度    
    - $1+2+2^2+....2^n=O(2^n)$
    - $1+\frac{1}{2}+\frac{1}{2^2}+....1/(2^n)=O(1)$
    - $1+2^2+3^2+...n^2=o(n^3)$
    - $1+\frac{1}{2}+\frac{1}{3}+...+\frac{1}{n}=O(log(n))$
